crowdfunder

This is a bootstrap of a personal crowd funding site built with Node.js using Express 4, Twitter Bootstrap, Font Awesome, Bootstrap Social Buttons

Originally forked from heroku/node-js-sample.
Started from the Startup Engineering Course on Coursera.

Live example on Heroku

I currently have it hosted on Heroku.
You can refer to http://mysterious-coast-9222.herokuapp.com/ for the live example.

Running Locally

Make sure you have Node.js and the Heroku Toolbelt installed.

git clone git@github.com:jian-wei/crowdfunder.git # or clone your own fork
cd crowdfunder
npm install
npm start

Your app should now be running on localhost:5000.

Deploying to Heroku

heroku create
git push heroku master
heroku open
